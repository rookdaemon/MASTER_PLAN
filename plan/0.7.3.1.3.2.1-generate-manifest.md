# 0.7.3.1.3.2.1 Generate Manifest [PLAN]

Produce the manifest file (`manifest.json`) that enumerates all files in the `traceability/` directory with their SHA‑256 checksums.

**Acceptance Criteria**
1. The manifest is a valid JSON file named `manifest.json`.
2. Each entry includes the relative path (e.g., `traceability/foo.txt`) and its SHA‑256 hash.
3. The manifest is written to the `build/` directory for later inclusion in the archive.
4. The generation script runs without external dependencies beyond the standard library.
5. Errors (e.g., missing files) cause the task to fail with a clear error message.
