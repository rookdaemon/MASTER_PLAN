# 0.7.3.1.1.3.3.2 Create git commit for traceability exports [PLAN]

Stage and commit the two export files in a single, atomic git commit.

**Acceptance Criteria**
1. The command `git add exports/traceability.md exports/traceability.json` succeeds without errors.
2. A git commit is created with the exact message “Export traceability package”.
3. The resulting commit diff contains **only** the two export files (no other paths).
4. The commit is created on the current branch without triggering a merge or rebase.
5. The commit hash is recorded (e.g., written to a temporary file) for later idempotency checks.
