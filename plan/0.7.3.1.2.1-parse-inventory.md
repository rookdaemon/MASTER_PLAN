# 0.7.3.1.2.1 Parse Inventory [PLAN]

Read `inventory.txt`, validate its format, and produce an ordered list of absolute file paths to be processed.

**Acceptance Criteria**
1. The task produces a JSON file `inventory-list.json` containing an array of absolute paths, preserving the order of `inventory.txt`.
2. Any duplicate or missing entries are logged, and the build fails if duplicates exist.
3. All listed paths are verified to exist on the filesystem; missing files cause a fatal error.
4. The JSON file is UTF‑8 encoded and passes a JSON schema validation step.
5. The task completes within 30 seconds for inventories up to 10 k entries.
