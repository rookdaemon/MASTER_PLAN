# 0.7.3.1.3.3.1.2 Compose Template [PLAN]

Draft the email body using the data gathered, adhering to the organization’s communication style guide.

#### Acceptance Criteria
1. Email body includes placeholders for repository link, documentation link, and each deliverable title/description.  
2. A “To” field is set to the primary stakeholder group; a “CC” field is auto‑populated from the stakeholder list.  
3. The template follows the approved style guide (tone, branding, signature).  
4. The draft is saved as “stakeholder‑notification‑draft.md” in the communications folder.  
5. The draft passes a lint/check against the organization’s email‑template validator with zero errors.
