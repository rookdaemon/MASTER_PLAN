# 0.7.3.1.1.3.1.1 Generate Provenance Table [PLAN]

Create the provenance table section of `artifacts/traceability/traceability.md`. The table must contain the columns **Deliverable**, **Author**, **Date**, **Version**, **Source‑Link** and include a row for every source artefact referenced in the overall traceability package.

**Acceptance Criteria**
1. The Markdown file `artifacts/traceability/traceability.md` exists after execution.  
2. The provenance table is syntactically correct Markdown and renders without errors.  
3. All required columns (Deliverable, Author, Date, Version, Source‑Link) are present and correctly labeled.  
4. Each source artefact appears as a separate row in the table.  
5. Dates are formatted in ISO‑8601 (`YYYY‑MM‑DD`).  
6. The updated `traceability.md` file is added to git and staged for commit.
