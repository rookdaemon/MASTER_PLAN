# 0.7.3.2.1.1.1.1.1.1 Check Existence [PLAN]
## Description
Implement a lightweight routine that checks whether `docs/codebase‑map.md` is present in the repository.

### Acceptance Criteria
1. The routine performs a filesystem check for the exact path `docs/codebase‑map.md`.  
2. Returns a boolean flag `exists` without loading file contents.  
3. Logs an informational message indicating the result of the check.  
4. Execution time is < 50 ms on a typical CI runner.  
5. The routine is pure (no side‑effects) and can be unit‑tested.
