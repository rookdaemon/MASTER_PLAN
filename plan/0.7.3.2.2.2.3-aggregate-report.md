# 0.7.3.2.2.2.3 Aggregate Report [PLAN]

Combine results from multiple vulnerability databases (Snyk and OSV) and produce a consolidated report.

**Acceptance Criteria**
1. Run OSV scanner (or API) against `artifacts/dependencies.json` and store output in `artifacts/osv-results.json`.
2. Merge `artifacts/snyk-results.json` and `artifacts/osv-results.json` into a single JSON file `artifacts/dependency-vuln-report.json`.
3. For each unique CVE, include fields: `cve_id`, `severity`, `affected_versions`, `source` (Snyk/OSV), and `remediation`.
4. Flag any CVE with severity **Critical** or **High** in a separate `artifacts/high‑risk-vulns.json`.
5. Add a summary section to `README.md` linking to the generated reports.
