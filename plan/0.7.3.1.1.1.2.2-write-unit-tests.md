# 0.7.3.1.1.1.2.2 Write Unit Tests [PLAN]

Create a test suite `tests/test_collect_artifacts.py` that:
- Sets up a temporary `artifacts/` hierarchy with at least two deterministic files (e.g., `artifacts/provenance/provenance.json` and `artifacts/config/example.yaml`).
- Executes `scripts/collect_artifacts.py`.
- Loads `temp/artifact_hashes.json` and asserts that both known files appear with correct SHA‑256 hashes.
- Verifies that provenance files are not present in the JSON output.
- Checks that `owner` defaults to `plan-guardian` when front‑matter is absent.

**Acceptance Criteria**
1. Tests can be run with `pytest -q` and all pass on a clean environment.
2. Hash values match expected pre‑computed SHA‑256 digests.
3. Test suite validates exclusion of provenance files.
4. Test suite confirms correct population of `created_at` and `owner` fields.
5. Coverage report shows ≥80 % of `collect_artifacts.py` exercised.
