# 0.7.3.1.1.3.1 Export markdown table of traceability [PLAN]

Generate `traceability.md` from `provenance.json` containing a markdown table with columns **ID**, **Type**, **Artifact(s)** (each artifact rendered as a relative hyperlink).

**Acceptance Criteria**
1. The script reads `provenance.json` and extracts every deliverable’s ID, type, and associated artifact paths.
2. The resulting `traceability.md` is written to `exports/traceability.md`.
3. The markdown table includes a header row and one data row per deliverable; no rows are missing or duplicated.
4. All artifact links are relative (e.g., `[artifact1](../artifacts/artifact1.png)`).
5. Running the script multiple times overwrites the file with identical content (idempotent).
