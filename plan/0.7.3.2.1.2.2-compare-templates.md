# 0.7.3.2.1.2.2 Compare Templates [PLAN]

For each artifact in the inventory, open the file and verify that its structure matches the approved template and versioning scheme.

**Acceptance Criteria**
1. Open every artifact in its native application and confirm no load errors.
2. Check that header sections, column headings, and required metadata fields exactly match the approved template definitions (v2.4).
3. Validate that the file name includes a version number following the `{major}.{minor}` pattern and matches the package release version.
4. Record a pass/fail result in the “Compliance Status” column of `artifact-inventory.xlsx`.
5. Document any template mismatches in a temporary note (e.g., `temp-mismatches.md`) for later consolidation.
