# 0.7.3.2.1.2 Check Template & Version Compliance [PLAN]

Validate that each artifact follows the prescribed template and versioning scheme outlined in the organization’s traceability standards (v2.4).

**Acceptance Criteria**
1. All artifacts open without errors in their native applications (Excel, PDF, etc.).
2. Header sections, column headings, and required metadata fields exactly match the approved templates.
3. Version numbers in file names follow the `{major}.{minor}` pattern and are consistent with the package’s release version.
4. Any deviations are logged in `template‑issues.md` with severity levels (minor/major).
5. A sign‑off note confirming template compliance is added to `template‑issues.md`.
