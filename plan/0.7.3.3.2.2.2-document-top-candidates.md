# 0.7.3.3.2.2.2 Document Top Candidates [PLAN]

## Description
Analyse the matrix from 0.7.3.3.2.2.1 and select at least two highest‑scoring sections. Produce a concise justification for each, referencing the scores.

## Acceptance Criteria
1. A markdown file `artifacts/0.7.3.3.2-top-candidates.md` lists the selected sections.
2. For each candidate, provide:
   * Section title
   * Total score (sum of the three criteria)
   * Brief justification (≤ 50 words) linking scores to narrative fit.
3. The document is committed to the `artifacts/` directory.
4. The selection process is traceable back to the matrix entries.
5. No changes are made to any other plan files.
