# 0.7.3.1.1.3.2.2 Enrich table with latest Git commit hash [PLAN]

**Description**  
Extend `scripts/generate-artifact-table.py` to obtain the latest git commit hash for each artifact file (`git log -n 1 --pretty=%H -- <file>`) and insert it into the **Last Commit Hash** column of the markdown table.

**Acceptance Criteria**
1. For every row, the script runs the git command against the artifact’s `relative_path` and captures the resulting hash.
2. If the git command fails (e.g., file not tracked), the script records `N/A` in the **Last Commit Hash** column and logs a warning to stderr.
3. The final `artifacts/INDEX_TABLE.md` contains all four required columns, with rows sorted alphabetically by **Category** then **Artifact Name**.
4. The script still exits with status 0 only when the table file is successfully written; otherwise it fails with a clear error.
5. Unit‑test‑style validation (e.g., a small test dataset) is documented in a comment block within the script.
