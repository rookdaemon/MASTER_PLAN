# 0.7.3.2.2.2.1 Build Dependency Parser [PLAN]

**Description**  
Create a Node.js script that walks `src/`, parses each `*.js` and `*.ts` file using an AST library (e.g., @babel/parser or TypeScript compiler API), and extracts imported module paths or require calls.

**Acceptance Criteria**
1. The script recursively scans the entire `src/` directory for `.js` and `.ts` files.  
2. For every file, the script identifies `import`, `export … from`, and `require()` statements.  
3. Extracted import paths are normalized to the repository’s module naming convention.  
4. The script outputs a JSON object mapping each source file path to a list of raw import strings.  
5. The script can be executed with `node tools/extract-deps.js` and exits with status 0 on success.
