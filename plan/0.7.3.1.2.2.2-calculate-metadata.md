# 0.7.3.1.2.2.2 Calculate Metadata [PLAN]

For each path supplied by the Load Inventory step, compute SHA‑256, MD5, file size, creation time, and modification time. Large files (> 1 GB) must be streamed.

**Acceptance Criteria**
1. Returns an object per file with keys: `path`, `sha256`, `md5`, `size_bytes`, `created_iso`, `modified_iso`.
2. SHA‑256 and MD5 values match those produced by external tools (e.g., `sha256sum`, `md5sum`).
3. Timestamps are converted to ISO‑8601 UTC strings.
4. Streaming implementation ensures memory usage < 100 MiB even for 10‑GB files.
5. The function `processFiles(paths: string[]): Promise<Metadata[]>` resolves when all files are processed, preserving input order.
