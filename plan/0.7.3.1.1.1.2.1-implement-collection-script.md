# 0.7.3.1.1.1.2.1 Implement Collection Script [PLAN]

Develop `scripts/collect_artifacts.py` that:
- Recursively walks `artifacts/`.
- Determines file type (`code` vs `config`) by extension.
- Reads front‑matter of code files to infer `owner` when present.
- Retrieves the Git commit date for each file (`git log -1 --format=%cI <path>`).
- Computes SHA‑256 hash.
- Skips any file under `artifacts/provenance/` to avoid circularity.
- Writes an array of JSON objects to `temp/artifact_hashes.json`.

**Acceptance Criteria**
1. Script runs without errors on a fresh checkout.
2. Output JSON contains a `hash` field for every non‑provenance file discovered.
3. Each entry includes the required fields (`id`, `type`, `source`, `created_at`, `owner`, `hash`).
4. Files in `artifacts/provenance/` are omitted from the output.
5. The script validates that `temp/` directory exists or creates it automatically.
