# 0.7.3.2.2.1.2 Assign Ownership to Each Directory [PLAN]

**Description**  
For every directory identified in sub‑task 0.7.3.2.2.1.1, consult the engineering org chart and stakeholder registry to allocate at least one owning team or individual.

**Acceptance Criteria**
1. A collaborative spreadsheet (`artifacts/ownership-assignment.xlsx`) is created with columns: `Directory`, `Owner(s)`.  
2. Each row (directory) has one or more owners populated (team name or email).  
3. The spreadsheet is reviewed and signed off by the corresponding module leads (recorded as a comment on the PR).  
4. A copy of the finalized assignments is exported to `artifacts/ownership-assignment.json`.  
5. The assignment file is version‑controlled on branch `inventory/assign-owners`.
