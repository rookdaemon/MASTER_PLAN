# 0.7.3.3.2.1.1 Write Extraction Script [PLAN]

## Description
Create a Python script `scripts/extract_section_outline.py` that reads `plan/0.7.3-plan-resilience-and-propagation.md`, scans for all top‑level H2 headings (`## `), records each heading’s exact text and the line number where it starts, and outputs a YAML structure.

## Acceptance Criteria
1. The script accepts an input markdown path and an output yaml path via command‑line arguments.
2. It correctly identifies every H2 heading, preserving case and punctuation.
3. The produced YAML matches the schema: a list of objects with `heading`, `text`, and `line`.
4. Running the script on the target file produces no traceback or errors.
5. The script is placed under `scripts/` and is executable (`chmod +x`).
