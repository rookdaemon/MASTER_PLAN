# 0.7.3.1.1.1.2.2 Record initial commit hash [PLAN]

Capture the SHA‑1 of the initial artifact‑structure commit and store it in documentation.

**Acceptance Criteria**
1. The script extracts the hash of the most recent commit (`git rev-parse HEAD`).  
2. `ARTIFACTS_SETUP.md` contains a top‑level heading `Initial Commit`.  
3. Under that heading the extracted hash is written on its own line, prefixed with `Hash: `.  
4. The heading and hash are appended atomically (no partial writes).  
5. The file is saved with UTF‑8 encoding and no trailing whitespace.
