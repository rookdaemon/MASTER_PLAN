# 0.7.3.2.2.1.2 Execute Analysis [PLAN]

Run the configured static analysis tools across the entire source tree, aggregate results, and export them in SARIF format.

**Acceptance Criteria**
1. All source files (`*.py`, `*.js`, etc.) are scanned by both Bandit and SonarQube in a single CI job.
2. The combined SARIF file (`artifacts/static-analysis.sarif`) contains entries from both tools.
3. Each finding includes a severity label (Critical, High, Medium, Low) as defined by the tools' mapping.
4. Execution time does not exceed the CI job timeout (e.g., 30 minutes) for the target codebase.
5. Any tool failures cause the CI job to fail with a clear error message.
