# 0.7.3.2.5.1.2 Send and Log Sign‑offs [PLAN]

Distribute the sign‑off request to each reviewer and the stakeholder, then record each received signature in a tracking sheet.

**Acceptance Criteria**
1. Email/template dispatched to every reviewer and the stakeholder using the approved distribution list.  
2. A shared Google Sheet (or equivalent) is created with columns: Reviewer, Role, Sent Timestamp, Received Timestamp, Signature Type, Link to Signed Document.  
3. Every incoming signed PDF/e‑signature is uploaded to the designated folder and linked in the tracking sheet.  
4. The tracking sheet is read‑only for all except the sign‑off coordinator.  
5. Completion status (Received / Pending) is visible at a glance.
