# 0.7.3.1.2.1.2.2 Detect Duplicates [PLAN]

Analyse the `absolutePaths` list, identify any duplicate absolute paths, and fail the build if duplicates exist.

**Acceptance Criteria**
1. Scans `absolutePaths` and builds a set of unique entries.
2. If duplicates are found, logs each duplicate path with its occurrence indices.
3. Triggers a fatal error that halts further processing.
4. If no duplicates, passes silently and outputs `uniquePaths` (order‑preserved).
5. Runs in ≤ 1 second for 10 k entries.
