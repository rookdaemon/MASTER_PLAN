# 0.7.3.1.2.3.1.2 Check Internal Links [PLAN]

Validate that all internal hyperlinks in the generated traceability artifacts resolve correctly.

**Acceptance Criteria**
1. `markdown-link-check TRACEABILITY.md` reports zero broken links.  
2. In `TRACEABILITY.html`, every intra‑document anchor (`href="#..."`) scrolls to a valid element when opened offline (can be verified with a headless browser or `wget --spider`).  
3. All cross‑references from design decisions to originating requirements are present as backlinks in the markdown.  
4. All test‑result sections list the associated source code files as links, and those links are valid.  
5. A short report (`link-check-report.txt`) is generated summarizing the checks and is committed to the repo.
