# 0.7.3.1.1.1.1 Collect Plan Files [PLAN]

Traverse the repository to locate every `plan/*.md` file (including ancestors and siblings) and record their identifiers, file paths, and owners.

**Acceptance Criteria**
1. A script `scripts/collect_plan_files.py` that returns a JSON array of objects with `id`, `type: "plan"`, `source` (file path), `created_at` (git commit date), and `owner` (extracted from front‑matter `owner` field or defaults to `plan‑guardian`).
2. The script runs without errors on the current repository layout.
3. Output is stored temporarily as `temp/plan_files.json` for downstream merging.
4. Unit test `tests/test_collect_plan_files.py` validates that at least the three known plan files (`root`, `0.7.3`, `0.7.3.1.1.1`) are present in the JSON.
5. Documentation added to `docs/provenance.md` under “Data Sources – Plan Files”.
