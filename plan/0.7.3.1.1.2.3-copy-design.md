# 0.7.3.1.1.2.3 Copy Design Artifacts [PLAN]

Copy or symlink all design‑related assets into `artifacts/design/`.

**Acceptance Criteria**

1. All UML diagrams, architecture PDFs, and design notes from the source location are present in `artifacts/design/`.  
2. Files are added to git and appear staged (`git status`).  
3. A `design/manifest.yaml` lists each file with its size and SHA‑256 hash.  
4. At least one file exists in the directory after execution.
