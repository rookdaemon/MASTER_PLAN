# 0.7.3.1.1.3.2.1 Generate Archive File [PLAN]

Create a compressed archive (`traceability-<tag>.zip` or `traceability-<tag>.tar.gz`) that contains the version‑tagged traceability matrix files and a `USAGE_GUIDE.md`.

### Acceptance Criteria
1. The archive is named exactly `traceability-<tag>.zip` (or `.tar.gz`) where `<tag>` matches the current release tag.  
2. The archive includes:
   * All `traceability-matrix.*` files for the tag.  
   * A `USAGE_GUIDE.md` describing matrix interpretation.  
3. Archive creation succeeds on both Linux and macOS CI runners without errors.  
4. The archive size is recorded and reported in the CI log.  
5. The generated archive is placed in a temporary staging directory ready for the next step.
