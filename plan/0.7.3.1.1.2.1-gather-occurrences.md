# 0.7.3.1.1.2.1 Gather Occurrences [PLAN]
## Description
Search the codebase (or documentation) for the target entry identified in the sibling “Identify Entry” task. Record each match’s line number or a stable textual anchor.

## Acceptance Criteria
1. All existing occurrences of the target entry are listed with precise line numbers or unique anchors.  
2. If no occurrence exists, record “Entry not found”.  
3. The list is captured in a structured format (e.g., JSON array or simple bullet list) ready for downstream formatting.  
4. No changes are made to source files.  
5. The result is stored in the task’s output artifact.
