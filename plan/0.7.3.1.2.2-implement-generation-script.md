# 0.7.3.1.2.2 Implement Generation Script [PLAN]

Develop `scripts/generate-traceability.py` that reads collected artifacts and the cross‑reference schema, then produces a linked markdown document and an HTML version.

**Acceptance Criteria**
1. The script parses all artifact files produced by `0.7.3.1.1-collect-artifacts`.  
2. It builds bidirectional links according to the schema and writes `traceability/plan-guardian/TRACEABILITY.md`.  
3. Running the script a second time produces identical output (idempotent).  
4. The script logs a summary of generated sections and any missing links.  
5. The script is added to `requirements.txt`/`pyproject.toml` and passes unit tests (`tests/test_generate_traceability.py`).
