# 0.7.3.2.4.2.2 Verify Module Entries [PLAN]

**Description**  
Cross‑reference each module entry in the traceability document against the dataset produced by task 0.7.3.2.4.1 (Collect Module Ownership and Dependency Data).

**Acceptance Criteria**
1. Every module name in the document appears exactly once in the collected dataset.  
2. Ownership fields match the owners listed in the dataset.  
3. No extra modules are present that are absent from the dataset.  
4. Discrepancies are written to `artifacts/0.7.3.2.4.2.2/discrepancies.csv`.  
5. The verification step exits with status code 0 only when all entries are consistent.
