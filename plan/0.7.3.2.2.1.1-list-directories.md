# 0.7.3.2.2.1.1 List Top-Level Source Directories [PLAN]

**Description**  
Scan the repository’s `src/` folder and produce a plain list of every immediate sub‑directory.

**Acceptance Criteria**
1. Execute a script that enumerates only first‑level entries under `src/`.  
2. The resulting list is saved as `artifacts/tmp-src-directories.txt`.  
3. The list contains no files, only directory names (e.g., `auth`, `payment`).  
4. The script logs its execution output and any errors to `artifacts/logs/list-directories.log`.  
5. The artifact is committed to the repo in a branch named `inventory/list-directories`.
