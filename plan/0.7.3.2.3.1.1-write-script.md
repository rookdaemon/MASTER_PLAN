# 0.7.3.2.3.1.1 Write Script [PLAN]

## Description
Create a portable Node.js (or Python) script named `tools/enumerate-modules.{js|py}` that:
* Walks the repository root `src/` directory.
* Lists only the immediate child directories.
* Emits the result as JSON (array of objects) and CSV (optional).

## Acceptance Criteria
1. The script can be executed via `npm run enumerate-modules` (or `python tools/enumerate-modules.py`) from the repository root.
2. Output format matches `{ "module": "name", "path": "src/name" }` for each entry.
3. No external OS‑specific dependencies; runs on Linux CI runners.
4. The script returns a non‑zero exit code if `src/` does not exist or is unreadable.
5. Unit tests (e.g., using Jest or pytest) confirm correct identification of modules in a mock `src/` tree.
