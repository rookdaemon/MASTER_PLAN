# 0.7.3.1.1.1.3.1 Implement Driver Script [PLAN]

Create the Python driver `scripts/build_provenance.py` that reads `temp/plan_files.json` and `temp/artifact_hashes.json`, merges them into a single ordered array, adds required metadata, and writes the result to `artifacts/provenance/provenance.json`.

**Acceptance Criteria**
1. The script exits with status 0 on success and prints a concise summary of the number of entries merged.
2. Output JSON contains a top‑level `generated_at` field (ISO‑8601 timestamp) and `generated_by` field set to `"plan-guardian"`.
3. The merged array is sorted alphabetically by the entry’s `id` field.
4. The script creates the directory `artifacts/provenance/` if it does not exist.
5. Unit test `tests/unit/test_build_provenance.py` verifies correct merging and metadata insertion.
