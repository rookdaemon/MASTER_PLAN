Update the project documentation to reference the new traceability archive location and name.

### Acceptance Criteria
1. The “Traceability Artifacts” section of the main README (or designated docs file) includes a bullet with the exact path:  
   `archives/traceability/traceability-<tag>.[zip|tar.gz]`.

4. The updated documentation builds successfully with the project's documentation pipeline.  
5. The doc change is reviewed and approved according to the project’s standard review process.
