# 0.7.3.1.1.3.1.1 Implement Collection Script [PLAN]

Create the Bash script `scripts/collect-artifact-list.sh` that recursively enumerates all non‑hidden files under `traceability/plan-guardian/artifacts/` and writes `artifacts/artifact-list.json` in the required format.

**Acceptance Criteria**
1. The script walks `traceability/plan-guardian/artifacts/` recursively, ignoring files/directories whose names begin with `.`.
2. For each file, the script determines a `category` (derived from the first‑level directory under `artifacts/`), the file `name` (basename), and the `relative_path` (path relative to `traceability/plan-guardian/artifacts/`).
3. The script outputs a JSON array of objects with fields `category`, `name`, `relative_path` to `artifacts/artifact-list.json`.
4. The script exits with status 0 on success; on any error it prints a clear message to stderr and exits with a non‑zero code.
5. The script header includes a short description, usage example, and required environment variables.
