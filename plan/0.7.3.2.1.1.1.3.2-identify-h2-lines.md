# 0.7.3.2.1.1.1.3.2 Identify H2 Lines [PLAN]
## Description
Scan the raw markdown content obtained from the previous step and locate every line that represents a Level‑2 heading (`## `).
## Acceptance Criteria
1. Each line that begins with exactly two hash symbols followed by a space is recognized as an H2 line.  
2. The line’s leading `## ` markup is stripped, preserving the remainder of the text unchanged.  
3. All identified H2 lines are collected in order of appearance into a list/array.  
4. Lines that are part of code fences or blockquotes are excluded from detection.  
5. The resulting list is passed to the next subtask without alteration.
