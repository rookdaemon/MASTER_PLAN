# 0.7.3.3.1.2.2.1 Review Mandatory Language [PLAN]
## Description
Inspect the governance statement for the presence of mandatory, prescriptive language (“shall,” “must,” “is required”) and ensure it conveys an ongoing management responsibility.

## Acceptance Criteria
1. All key obligations are expressed with mandatory verbs (shall, must, is required).  
2. Each obligation clearly indicates an ongoing, repeatable responsibility rather than a one‑off action.  
3. No weakening qualifiers (e.g., “may,” “could”) are used for mandatory requirements.  
4. Findings are documented in a brief note attached to the plan file.  
5. If deficiencies are found, they are highlighted for correction in the next sub‑task.
