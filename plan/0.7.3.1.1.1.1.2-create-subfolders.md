# 0.7.3.1.1.1.1.2 Create Sub‑folders [PLAN]

Inside `traceability/plan-guardian/artifacts/` create the sub‑folders `requirements/`, `design/`, `code/`, and `tests/`.

**Acceptance Criteria**
1. All four sub‑folders exist directly under `traceability/plan-guardian/artifacts/`.  
2. Each sub‑folder is created only if it does not already exist (idempotent).  
3. No additional files or directories are created at this step.  
4. Permissions inherit from the parent folder.  
5. A log entry `SUBFOLDERS_CREATED` is appended to `ARTIFACTS_SETUP.md`.
