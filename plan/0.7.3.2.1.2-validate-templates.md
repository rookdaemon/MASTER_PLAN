# 0.7.3.2.1.2 Validate Templates and Versioning [PLAN]

Check that each artifact adheres to the prescribed template structure and carries the correct version identifier as per the organization’s template registry.

**Acceptance Criteria**
1. Every artifact’s content matches the corresponding template layout.
2. Version numbers are present and follow the `vX.Y` scheme defined in the standards.
3. Template deviations are logged as failures in `review-checklist.md`.
4. A summary table of artifact → template compliance is produced.
5. All compliant artifacts are marked as “pass” in the checklist.
