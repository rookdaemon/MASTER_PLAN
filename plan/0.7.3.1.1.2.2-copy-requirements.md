# 0.7.3.1.1.2.2 Copy Requirement Artifacts [PLAN]

Copy or symlink every requirement document into `artifacts/requirements/`.

**Acceptance Criteria**

1. All files listed in `docs/requirements/` (or equivalent source) are present in `artifacts/requirements/`.  
2. No duplicate filenames exist; name collisions are resolved by renaming with a numeric suffix.  
3. All copied files are tracked by git (`git add` and `git status` shows them as staged).  
4. A simple checksum (e.g., `sha256sum`) is recorded in `artifacts/requirements/manifest.txt`.
