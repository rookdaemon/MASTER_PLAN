# 0.7.3.2.1.2 Check Templates & Versions [PLAN]

Ensure each artifact follows the prescribed template and carries the correct version number according to the organization’s standards.

**Acceptance Criteria**
1. Templates of all artifacts match the latest approved versions (v2.0 or later).
2. Version numbers are present in file headers and follow the `vX.Y` format.
3. Discrepancies in template or version are logged with file path and required correction.
4. A compliance matrix summarizing template/version status is saved as `template-version-matrix.md`.
5. No artifact is left unverified for template and version compliance.
