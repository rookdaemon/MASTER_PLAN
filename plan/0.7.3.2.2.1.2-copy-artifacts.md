# 0.7.3.2.2.1.2 Copy Artifacts [PLAN]

**Description:**  
Using the approved checklist, copy each artifact from its source location to `tmp/traceability_stage/`, preserving original filenames and directory structure where feasible.

**Acceptance Criteria**
1. Every artifact marked *Ready* in `traceability-checklist.md` exists in `tmp/traceability_stage/` with the same filename.
2. Copy operation logs each successful transfer with timestamp.
3. If a source file is missing, the log records the artifact name, expected source, and a brief reason; the artifact remains marked *Missing* in the checklist.
4. The staging directory contains **only** the copied artifacts (no extra files or folders).
5. After copying, a checksum (SHA‑256) file `stage_checksums.txt` is generated for all staged artifacts.
