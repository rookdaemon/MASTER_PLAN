# 0.7.3.1.1.3.3 Write INDEX.md File [PLAN]

Combine the commit‑hash header, the generated table, and a checklist into the final `traceability/plan-guardian/artifacts/INDEX.md`.

**Acceptance Criteria**
1. A script (`scripts/write-index.py`) creates `traceability/plan-guardian/artifacts/INDEX.md`.
2. The file starts with a line `Latest commit hash: <HASH>` where `<HASH>` is the most recent commit affecting any file under `artifacts/`.
3. Immediately after the header, the script inserts the contents of `artifacts/INDEX_TABLE.md`.
4. A checklist section is added at the end with a ticked box: `- [x] No missing references`.
5. CI runs a validation step that parses `INDEX.md`, confirms the header hash matches `git rev-parse HEAD` for the `artifacts/` directory, and verifies all listed Relative Paths exist.
