# 0.7.3.1.1.2 Generate Cross‑Reference Entries in Parent Plan [PLAN]

Insert cross‑reference entries into `plan/0.7.3-plan-resilience-and-propagation.md` that link each deliverable listed in the provenance package to its supporting artifact(s).

**Acceptance Criteria**
1. For every entry in `provenance.json`, a markdown bullet appears under a “## Deliverables” section in the parent plan, formatted as `- **{ID}** – {type} – [link to artifact]`.
2. All links resolve to files inside the repository (`artifacts/...` or `plan/...`).
3. The parent plan file is updated atomically and a commit message “Add traceability cross‑references” is recorded.
4. A summary table of the cross‑references is also added to `docs/traceability-summary.md`.
5. No duplicate entries are introduced; the process is idempotent.
