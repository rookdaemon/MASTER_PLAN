# 0.7.3.1.2.1.2 Identify Mapping Line [PLAN]
## Description
Search the opened `docs/codebase-map.md` for the first occurrence of the string `src/plan-guardian/`, record its line number, and capture the full line text.

## Acceptance Criteria
1. The search returns exactly one line containing `src/plan-guardian/`; if multiple matches exist, the first is used.  
2. The line number (1‑based) is stored in a variable `mapping_line_number`.  
3. The exact line content is stored in a variable `mapping_line_content`.  
4. No modifications are made to `docs/codebase-map.md`.  
5. The line number and content are printed to the console and written to a temporary JSON artifact `artifacts/mapping_line.json` with the shape `{ "line": <number>, "content": "<string>" }`.
