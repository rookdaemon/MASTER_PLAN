# 0.7.3.2.2.1.1.1.3 Format List & Add to Documentation [PLAN]
## Description
Combine the field names and their descriptions into a properly formatted Markdown bullet list and insert it into both the draft template and the reference document `docs/template-fields.md`.

## Acceptance Criteria
1. The final bullet list contains exactly the fields and descriptions from the prior subtasks.
2. The list is syntactically valid Markdown (each line begins with `- `).
3. The list is inserted into the placeholder section of the draft template file without breaking existing content.
4. The same list is added to `docs/template-fields.md` under a heading “Mapping Entry Fields”.
5. A short verification comment is appended to both files confirming successful insertion.
