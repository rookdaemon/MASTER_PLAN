# 0.7.3.2.2.3.1 Generate Checksums [PLAN]

Create SHA‑256, SHA‑384, and SHA‑512 checksum files for every built artifact.

**Acceptance Criteria**
1. A script (`scripts/generate-checksums.sh`) iterates over all files in `artifacts/build/` and produces `.sha256`, `.sha384`, and `.sha512` files.
2. Each checksum file contains the hash followed by two spaces and the artifact filename (standard `sha*sum` format).
3. All checksum files are placed under `artifacts/integrity/`.
4. The script exits with status 0 only when every artifact has a corresponding three checksum files.
5. The script logs the list of generated files to `artifacts/integrity/checksum-generation.log`.
