# 0.7.3.2.4.3.1 Compress Staged Artifacts [PLAN]

**Description:**  
Create a reproducible archive of the staged release artifacts produced by `0.7.3.2.4.1`, preserving permissions and timestamps.

**Acceptance Criteria**
1. The staging directory path is discovered from the output of `0.7.3.2.4.1`.
2. The archive `dist/plan-guardian-0.7.3.${DATE}.tar.gz` is created using `tar --preserve-permissions --gzip`.
3. Archive creation logs the exact source paths and resulting file size.
4. The archive file exists and is readable after creation.
5. Failure to create the archive aborts the step and writes an error entry to `logs/release-0.7.3.2.log`.
