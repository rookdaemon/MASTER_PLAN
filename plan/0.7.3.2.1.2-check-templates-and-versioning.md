# 0.7.3.2.1.2 Check Templates and Versioning [PLAN]

Validate that each artifact follows the prescribed template and carries the correct version identifier as defined in the standards document (v2.4).

**Acceptance Criteria**
1. Each artifact matches its template schema (column headers, required fields, format).
2. The version identifier in the filename matches the version declared inside the document header.
3. Any deviations from the template are logged with file name and description.
4. All version numbers are ≥ the minimum required version specified in the standards.
5. A summary table of template compliance is produced.
