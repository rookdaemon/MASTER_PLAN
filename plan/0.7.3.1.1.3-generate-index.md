# 0.7.3.1.1.3 Generate Artifact Index [PLAN]

Create `INDEX.md` that lists every collected artifact with relative paths and records the latest git commit hash.

**Acceptance Criteria**

1. `traceability/plan-guardian/artifacts/INDEX.md` exists.  
2. The index contains a markdown table with columns: Category, Artifact Name, Relative Path, Last Commit Hash.  
3. The latest commit hash of the entire `artifacts/` directory is recorded at the top of the file.  
4. The index is validated by a CI job that parses the table and confirms each listed file exists.  
5. A checklist section in the index confirms “No missing references” with a ticked box.
