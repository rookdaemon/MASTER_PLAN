# 0.7.3.1.1.3.1 Collect Artifact List [PLAN]

Enumerate every file placed under `traceability/plan-guardian/artifacts/` (excluding hidden files) and output a structured list for downstream processing.

**Acceptance Criteria**
1. A script (`scripts/collect-artifact-list.sh`) enumerates all non‑hidden files recursively under `traceability/plan-guardian/artifacts/`.
2. The script outputs a JSON file `artifacts/artifact-list.json` containing an array of objects with fields: `category`, `name`, `relative_path`.
3. The script exits with status 0 on success and non‑zero on failure, providing an error message on stderr.
4. CI can invoke the script and verify that `artifact-list.json` is non‑empty and well‑formed JSON.
5. The script is documented in its header with usage examples.
