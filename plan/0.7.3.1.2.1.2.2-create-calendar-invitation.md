# 0.7.3.1.2.1.2.2 Create Calendar Invitation [PLAN]

Using the approved draft, create the official calendar event in the organization’s calendar system (e.g., Google Calendar, Outlook).

### Acceptance Criteria
1. Event title, date, start/end times, and time‑zone match the approved draft exactly.  
2. The event description contains the full agenda and a clickable link to the traceability package.  
3. A video‑conference link (or physical room details) is generated and attached.  
4. The event is saved as a “Busy” entry to block the time slot for all invitees.  
5. The unique event ID (or meeting link) is recorded in the review documentation (`docs/review/meeting-log.md`).
