# 0.7.3.1.1.3.3.2 Sign Checksum with GPG [PLAN]

Create a GPG detached signature for the checksum file.

### Acceptance Criteria
1. The project’s GPG private key (identified by its fingerprint in the repo config) is used.  
2. Command `gpg --armor --output traceability-checksum.sha256.asc --detach-sign traceability-checksum.sha256` executes successfully.  
3. The resulting `traceability-checksum.sha256.asc` file is placed in `archives/traceability/`.  
4. Signature verification (`gpg --verify traceability-checksum.sha256.asc traceability-checksum.sha256`) succeeds as part of the task.  
5. If the private key is unavailable or the signing fails, the process aborts with a clear error message.
