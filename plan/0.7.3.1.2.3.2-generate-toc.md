# 0.7.3.1.2.3.2 Generate TOC [PLAN]

Create a Markdown Table of Contents that links to each file’s entry within `TRACEABILITY-MANIFEST.md` using anchor links derived from file names.

**Acceptance Criteria**
1. A TOC entry is generated for every file listed in `inventory.txt`.  
2. Each entry links to the corresponding section in the manifest using a valid Markdown anchor (`[link](#anchor)`).  
3. The TOC is placed at the very top of `TRACEABILITY-MANIFEST.md`.  
4. Anchor generation is deterministic and handles spaces, special characters, and case‑insensitivity.  
5. A test validates that all TOC links resolve to existing headings in the final manifest.
