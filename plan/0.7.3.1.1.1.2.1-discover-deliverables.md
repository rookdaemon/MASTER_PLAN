# 0.7.3.1.1.1.2.1 Discover Deliverables [PLAN]

Implement logic that walks `artifacts/` and `plan/` directories, identifying files that constitute deliverables.

**Acceptance Criteria**
1. A function `isDeliverable(filePath)` returns true when a file has front‑matter `deliverable: true` **or** matches a naming pattern (e.g., `*-artifact.*`, `*-plan.md`).
2. The walk recursively captures all matching files and returns an array of absolute and relative paths.
3. Unit tests cover at least three positive and two negative cases for `isDeliverable`.
4. The discovery module exports a promise‑based API `getDeliverables(): Promise<string[]>`.
5. Execution time for a repository of ~500 files is under 2 seconds on typical CI hardware.
