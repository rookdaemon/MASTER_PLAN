# 0.7.3.3.2.2 Evaluate Candidate Locations [PLAN]

## Description
Using the outline produced in 0.7.3.3.2.1, assess each top‑level section for suitability as the insertion point for the cross‑reference. Criteria include visibility, relevance, and minimal disruption to existing narrative flow.

## Acceptance Criteria
1. A matrix `artifacts/0.7.3.3.2-candidate-evaluation.yaml` ranks each H2 section by the three criteria (visibility, relevance, disruption) on a 1‑5 scale. *(produced by subtask 0.7.3.3.2.2.1)*
2. At least two top‑ranked candidate sections are documented with justification. *(produced by subtask 0.7.3.3.2.2.2)*
3. The evaluation process is reproducible via a short script or notebook. *(produced by subtask 0.7.3.3.2.2.3)*
4. All artifacts are committed to the `artifacts/` directory.
5. No modifications are made to `plan/0.7.3-plan-resilience-and-propagation.md` at this stage.
