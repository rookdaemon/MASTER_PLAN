# 0.7.3.1.1.3.2 Export JSON traceability copy [PLAN]

Create a pretty‑printed copy of `provenance.json` named `traceability.json` in the `exports/` folder.

**Acceptance Criteria**
1. The task copies `provenance.json` to `exports/traceability.json`.
2. The JSON is formatted with an indentation of 2 spaces (pretty‑printed).
3. File permissions allow read/write for repository contributors.
4. If `exports/traceability.json` already exists with identical content, the task does not modify the file (idempotent).
5. The resulting JSON passes a JSON schema validation step (schema provided elsewhere).
