# 0.7.3.1.2.1.2.3 Verify Existence [PLAN]

For each path in `uniquePaths`, confirm the file exists on disk; report missing files and abort on any failure.

**Acceptance Criteria**
1. Uses `fs.access(path, fs.constants.F_OK)` (or equivalent) for each path.
2. Collects any missing paths, logs them with clear error messages.
3. If any missing files are detected, terminates with a fatal error.
4. On success, outputs `uniqueExistingPaths` (identical to `uniquePaths`).
5. Completes within 5 seconds for 10 k entries.
