# 0.7.3.2.2.2.2 Compute SHA‑256 Checksum [PLAN]

**Description:**  
Calculate the SHA‑256 checksum of the archive produced in the previous subtask and write it to a file named `traceability-0.7.3.1-20260409.zip.sha256` in the same delivery directory.

**Acceptance Criteria**
1. The checksum is computed using a standard SHA‑256 tool (e.g., `sha256sum` or `shasum -a 256`).
2. The checksum file contains a single line with the checksum followed by the archive filename.
3. The checksum value matches a re‑computed checksum of the archive (verification step).
4. The `.sha256` file is placed in `deliverables/traceability/` and is readable.
5. The operation logs a success message; any mismatch aborts the workflow.
