# 0.2.2.4.1 Bio-Synthetic Interface Design [DONE]
## Description

Design and implement functional interfaces between biological neural tissue and synthetic computational substrates. This is the foundational hardware/wetware layer that makes hybrid cognition possible.

### Key challenges
- Signal transduction between biological electrochemical signaling and synthetic digital/analog computation
- Biocompatibility — interfaces must not damage or degrade biological tissue over time
- Bandwidth — interfaces must support sufficient information throughput for real-time cognitive integration
- Bidirectional communication — synthetic components must both read from and write to biological neural circuits

### Scope boundaries
- This card covers the interface layer only — not the synthetic substrate itself (see 0.2.1) or the brain model (see 0.2.2.1)
- Unified consciousness across the interface boundary is handled by sibling 0.2.2.4.2
- Failure-mode resilience at the interface is handled by sibling 0.2.2.4.3

## Decision

### D1 — Electrode Technology Selection

**Context:** The interface must make physical contact with biological neural tissue for both reading and writing signals. Multiple electrode technologies exist with different trade-offs.

**Options Considered:**
1. **Penetrating microelectrode arrays (Utah-style)** — rigid silicon probes inserted into cortical tissue. High single-neuron resolution but cause tissue damage and glial scarring over time.
2. **Flexible polymer-based electrode arrays** — soft substrates (parylene-C, polyimide) with conducting polymer contacts (PEDOT:PSS). Match tissue elasticity, reducing chronic immune response. Lower initial channel density but better long-term stability.
3. **Neural dust (ultrasonic motes)** — wireless sub-mm implants powered by ultrasound. No tethering, minimal tissue disruption, but limited per-mote bandwidth and unproven at scale.

**Choice:** Flexible polymer-based electrode arrays (Option 2) as the primary modality, with the architecture explicitly designed (Layer 1/2 separation) to allow future swap to neural dust or other technologies without changing higher layers.

**Rationale:** Biocompatibility over multi-year timescales is a hard requirement. Flexible substrates matching brain tissue elasticity (~1–10 kPa) are the only proven approach for chronic implantation without progressive signal degradation. The modular architecture mitigates the risk of choosing wrong by making the electrode layer replaceable.

**Consequences:** Lower initial electrode density than rigid arrays (target ≥10,000/mm² may require iterative improvement). Fabrication complexity is higher. But chronic stability is non-negotiable for a system intended to support continuous hybrid cognition.

### D2 — Safety Enforcement Layer

**Context:** Electrical stimulation of neural tissue carries risks: seizure induction, tissue damage from excessive charge, off-target activation. The system must enforce safety limits.

**Options Considered:**
1. **Software-only safety** — all limits enforced in firmware/software with watchdog timers. Flexible, updatable, but vulnerable to software bugs or corruption.
2. **Hardware-enforced safety with software supervision** — charge density limits and seizure detection circuits implemented in analog/mixed-signal hardware that cannot be overridden by software. Software adds additional policy on top.

**Choice:** Hardware-enforced safety with software supervision (Option 2).

**Rationale:** Neural tissue damage from a safety failure is irreversible. Hardware limits provide a fail-safe floor that persists even if software crashes, is corrupted, or has bugs. Software can restrict further (e.g., lower limits for sensitive regions) but cannot exceed hardware maximums. Defense-in-depth principle.

**Consequences:** Hardware safety circuits are less flexible — changing limits requires physical modification. This is acceptable because the limits are based on well-established tissue damage thresholds that change slowly if at all.

### D3 — Calibration Strategy

**Context:** Biological neural tissue reorganizes over time (plasticity), and electrode impedance drifts. The mapping between electrode signals and neuron identities must be maintained.

**Options Considered:**
1. **Static calibration** — one-time mapping at implantation, with periodic manual recalibration sessions. Simple but requires downtime and degrades between sessions.
2. **Continuous closed-loop adaptive calibration** — the interface continuously adjusts read/write mappings based on observed neural responses, running as a background process at Layer 3.

**Choice:** Continuous closed-loop adaptive calibration (Option 2).

**Rationale:** Hybrid cognition must operate continuously without interruption for recalibration. Neural plasticity is ongoing — the brain rewires in response to the interface itself. Only continuous adaptation can maintain fidelity over operational timescales. The stimulation-response model at Layer 3 adapts its write pathway based on measured read responses, forming a self-correcting loop.

**Consequences:** Higher computational overhead at Layer 3. Risk of calibration instability (oscillation) requires damping mechanisms. Calibration model must be validated against ground truth periodically.

## Contracts

### Interface: `RawNeuralSignal` (Layer 1 → Layer 2)

**Preconditions:**
- Electrode array is physically implanted and impedance-verified
- Electrode channel is not flagged as failed by health monitoring

**Postconditions:**
- Continuous analog voltage trace delivered at native sampling rate per channel
- Signal includes both spike activity and LFP components (no filtering at this layer)

**Invariants:**
- Signal amplitude reflects true extracellular potential within electrode noise floor
- Channel identity is stable — same physical electrode always maps to same channel ID

### Interface: `SpikeTrainStream` (Layer 2 → Layer 3)

**Preconditions:**
- Raw signal has been amplified, filtered (300 Hz – 6 kHz bandpass), and digitized (≥30 kHz, ≥12-bit)
- Spike detection threshold has been set (per-channel, adaptive)

**Postconditions:**
- Stream of time-stamped spike events, each tagged with a putative neuron ID (from spike sorting)
- Spike waveform snippet (≤2 ms window) attached for downstream validation
- Latency from extracellular spike to emitted event: < 100 μs

**Invariants:**
- Neuron IDs are consistent within a calibration epoch (may remap across epochs)
- No spike event is emitted without crossing detection threshold

### Interface: `StimulationCommand` (Layer 3 → Layer 2)

**Preconditions:**
- Target neuron population specified in brain-atlas coordinates
- Desired activation pattern specified (spike rate, timing)
- Command passes Layer 3 safety envelope check (no seizure-risk patterns)

**Postconditions:**
- Charge-balanced biphasic current pulses delivered to the electrode subset mapped to target population
- Charge density per phase ≤ hardware safety limit
- Latency from command receipt to current delivery: < 200 μs

**Invariants:**
- Hardware safety limits cannot be exceeded regardless of command content
- Stimulation is always charge-balanced (net zero charge per pulse)

### Interface: `HybridCognitionBus` (Layer 4 → external consumers)

**Preconditions:**
- At least one physical interface unit is operational and calibrated
- Integration Gateway has completed initial registration of all active arrays

**Postconditions:**
- Unified bidirectional neural data stream presented to consumers
- Biological neural state readable as `NeuralStateSnapshot` at configurable temporal resolution
- Synthetic activation requests accepted as `SyntheticActivationRequest` and dispatched to appropriate physical arrays
- Per-region health and bandwidth metrics available via `InterfaceHealthReport`

**Invariants:**
- Logical interface is stable regardless of number of underlying physical arrays
- Loss of a physical array triggers degradation signal (to 0.2.2.4.3) but does not crash the bus
- Bandwidth allocation is non-starvation: every active region gets minimum guaranteed throughput

## Threshold Registry

| Name | Value | Unit | Valid Range | Rationale | Sensitivity |
|------|-------|------|-------------|-----------|-------------|
| Electrode density | 10,000 | electrodes/mm² | 1,000–100,000 | Single-neuron resolution requires electrode spacing ≤ ~10 μm; cortical column coverage requires mm-scale arrays | Below 1,000: lose single-neuron resolution. Above 100,000: fabrication limits with current polymer technology |
| Amplifier input noise | 5 | μV_rms | 1–10 | Extracellular spikes are 50–500 μV; SNR ≥ 10 dB requires noise ≤ ~50 μV, with margin | Above 10: spike detection degrades. Below 1: diminishing returns, cost increases |
| ADC sampling rate | 30 | kHz/channel | 20–100 | Nyquist for 6 kHz spike bandwidth = 12 kHz minimum; 30 kHz gives 2.5× oversampling for waveform fidelity | Below 20: aliasing artifacts. Above 100: data rate becomes prohibitive at scale |
| ADC resolution | 12 | bits | 10–16 | Dynamic range covers both LFP (~1 mV) and spikes (~500 μV) against noise floor | Below 10: quantization noise dominates. Above 16: diminishing returns |
| Read pathway latency | 100 | μs | 50–500 | Biological synaptic delay is ~1 ms; interface must add < 10% to stay transparent | Above 500: perceptible delay in closed-loop cognition. Below 50: difficult with current ADC technology |
| Write pathway latency | 200 | μs | 100–1,000 | Combined with read latency, total round-trip must stay < 500 μs | Above 1,000: round-trip exceeds synaptic timing budget |
| Round-trip latency | 500 | μs | 200–1,000 | Must not exceed biological synaptic timing (~1 ms) to maintain cognitive integration | Above 1,000: hybrid cognition perceived as "laggy," unity breaks down |
| Charge density limit | 30 | μC/cm²/phase | 10–50 | Shannon limit for safe stimulation; above ~40 μC/cm² causes irreversible tissue damage | Above 50: tissue necrosis. Below 10: insufficient to activate neurons |
| Spike detection TPR | 95 | % | 90–99 | Missed spikes = lost information; below 90% cognitive fidelity degrades noticeably | Below 90: significant information loss. Above 99: requires impractical SNR |
| Spike detection FPR | 5 | % | 1–10 | False spikes inject noise into synthetic model; above 10% corrupts neural state | Above 10: synthetic side gets unreliable input. Below 1: threshold too conservative, loses real spikes |
| Spike sorting accuracy | 90 | % | 80–99 | Mis-attributed spikes confuse neuron identity in brain model | Below 80: neuron identity unreliable. Above 95: requires very high SNR |
| Stimulation spatial precision | 90 | % on-target | 80–99 | Off-target activation corrupts intended neural state changes | Below 80: stimulation is too diffuse for precise cognition. Above 99: requires single-synapse targeting |
| Biocompatibility duration | 5 | years | 1–20 | Interface must outlast brain plasticity adaptation period; replacement surgery carries risk | Below 1: impractical maintenance cycle. Above 20: unproven with any current material |
| Signal degradation threshold | 20 | % max degradation | 5–50 | Defines when an electrode is considered "failed" and triggers degradation protocol | Below 5: too aggressive, healthy electrodes flagged. Above 50: unacceptable fidelity loss |
| Cortical column bandwidth | 200 | Kbps (compressed) | 50–1,000 | After spike compression, sufficient for ~10,000 neurons at ~10 Hz avg firing rate | Below 50: lossy compression degrades fidelity. Above 1,000: over-provisioned |

## Behavioral Spec

### Scenario: Read pathway — spike detection and streaming

```
Given a biological neuron fires an action potential within range of an electrode
When the extracellular voltage crosses the spike detection threshold
Then a SpikeTrainStream event is emitted within 100 μs
  And the event contains: timestamp, putative neuron ID, waveform snippet
  And the event is delivered to Layer 3 for coordinate mapping
```

### Scenario: Write pathway — targeted stimulation

```
Given the synthetic substrate issues a SyntheticActivationRequest for neuron population P
When Layer 3 translates the request to electrode coordinates and passes safety checks
Then Layer 2 delivers charge-balanced biphasic current pulses to the mapped electrodes within 200 μs
  And charge density per phase does not exceed 30 μC/cm²
  And only electrodes mapped to population P are activated (≤10% off-target)
```

### Scenario: Safety interlock — seizure detection

```
Given the interface is operating normally
When Layer 3 detects a seizure-like activity pattern (sustained high-frequency synchronous firing > 2 standard deviations above baseline across ≥30% of channels)
Then all stimulation commands are immediately suspended
  And an alert is propagated via InterfaceHealthReport
  And stimulation remains suspended until the seizure-like pattern subsides for ≥5 seconds
  And the event is logged with full neural state for post-hoc analysis
```

### Scenario: Adaptive calibration — electrode drift compensation

```
Given the interface has been operating for > 24 hours since last calibration epoch
When Layer 3 detects that spike sorting accuracy has dropped below 85% for a region
Then a background recalibration is triggered for that region
  And the recalibration uses recent stimulation-response pairs to update the electrode-to-neuron mapping
  And the read/write pathways continue operating with the previous mapping until the new mapping is validated
  And the new mapping is atomically swapped in once validation confirms accuracy ≥ 90%
```

### Scenario: Graceful array loss

```
Given the Integration Gateway is managing N physical interface arrays
When one array's InterfaceHealthReport indicates signal degradation > 50% across ≥80% of its channels
Then the Gateway marks that array as degraded
  And notifies the graceful degradation layer (0.2.2.4.3) via InterfaceHealthReport
  And redistributes bandwidth allocation among remaining arrays
  And the HybridCognitionBus continues operating with reduced coverage
```

## Acceptance Criteria
- **Signal transduction**: Bidirectional signal conversion demonstrated — biological electrochemical signals faithfully encoded into synthetic digital/analog signals and vice versa, with defined fidelity metrics
  - Given/When/Then: see Behavioral Spec "Read pathway" and "Write pathway" scenarios
- **Biocompatibility**: Interface materials and mechanisms validated for extended contact with neural tissue without inflammation, scarring, or signal degradation over operational lifetime (≥5 years, ≤20% degradation)
- **Bandwidth**: Interface throughput and latency characterized; must operate within biological neural timing constraints (read < 100 μs, write < 200 μs, round-trip < 500 μs; cortical-column bandwidth ≥ 200 Kbps compressed)
- **Read pathway** (biological→synthetic): Reliable recording of neural spike trains and LFPs with single-neuron resolution (≥95% TPR, ≤5% FPR, ≥90% spike sorting accuracy)
- **Write pathway** (synthetic→biological): Precise stimulation of targeted neural populations (≥90% on-target, charge density ≤ 30 μC/cm²/phase) without off-target activation or tissue damage
  - Given/When/Then: see Behavioral Spec "Safety interlock" scenario
- **Scalability**: Architecture supports scaling from single cortical column (~10K neurons) to cortical-region-scale (~1M neurons) via modular array deployment without architectural changes
- **Adaptive calibration**: Interface maintains fidelity despite neural plasticity and electrode drift via continuous closed-loop calibration
  - Given/When/Then: see Behavioral Spec "Adaptive calibration" scenario
- **Fault tolerance**: Loss of individual electrode arrays triggers graceful degradation, not system failure
  - Given/When/Then: see Behavioral Spec "Graceful array loss" scenario
