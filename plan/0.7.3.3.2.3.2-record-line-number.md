# 0.7.3.3.2.3.2 Record Line Number [PLAN]

## Description
Determine the exact line number (or unique marker) where the cross‑reference will be inserted immediately after the selected header.

## Acceptance Criteria
1. The line number is calculated accurately using a simple line‑count script or manual count.
2. The line number is stored in `artifacts/0.7.3.3.2-selected-point.yaml` under key `line_number`.
3. The line number refers to a location that is unique within `plan/0.7.3-plan-resilience-and-propagation.md`.
4. The artifact file contains both `section_header` (from subtask .1) and `line_number`.
5. No changes are applied to the target plan file at this point.
