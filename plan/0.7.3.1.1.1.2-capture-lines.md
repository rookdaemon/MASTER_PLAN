# 0.7.3.1.1.1.2 Capture Entry Lines [PLAN]
## Description
Search the in‑memory contents of `docs/codebase-map.md` for every occurrence of the string `src/plan-guardian/`. Record each match’s line number and a short surrounding text snippet (or a unique anchor) to uniquely identify the location.

## Acceptance Criteria
1. All matches of `src/plan-guardian/` are identified; the count matches the actual occurrences in the file.  
2. For each match, the exact line number is recorded.  
3. A surrounding text snippet (e.g., 30 characters before and after) is captured for each match.  
4. If no matches are found, the result explicitly states “not found”.
