# 0.1.2.5 Ethical Frameworks Updated [DONE]
## Description

F2.5 — Update ethical frameworks to include non-biological conscious entities. Existing ethical systems (utilitarian, deontological, virtue ethics, care ethics, etc.) must be extended or revised to account for machine consciousness, including questions of moral status, moral patiency, and obligations toward non-biological minds.

This card produces a document (`docs/ethical-frameworks-for-non-biological-consciousness.md`) that:
1. Surveys major ethical traditions and identifies where they implicitly assume biological substrates.
2. Proposes substrate-independent extensions for each tradition.
3. Defines moral status criteria applicable to any conscious entity regardless of substrate.
4. Establishes concrete ethical guidelines for the lifecycle of conscious systems (creation, modification, suspension, termination).

Depends on 0.1.2.3 (Philosophical Objections Addressed) because the extensions must incorporate responses to zombie arguments, Chinese Room, etc.

## Decision: Document Structure and Framework Coverage

### Context
F2.5 requires updating ethical frameworks to include non-biological conscious entities. The output is a single self-contained document. Key choices: which frameworks to cover, how to structure the analysis, and how to define moral status criteria.

### Options Considered
1. **Framework-by-framework analysis with unified moral status criteria** — Audit each tradition for biological bias, propose substrate-independent extensions, then synthesize convergent moral status criteria across all four.
2. **Topic-organized analysis** — Organize by ethical topic (moral status, lifecycle ethics, obligations) and discuss each framework within each topic section.

### Choice
Option 1: Framework-by-framework analysis with unified moral status criteria.

### Rationale
Preserves the internal logic of each tradition, making extensions traceable to their philosophical roots. A separate convergence section (Section 6) then identifies cross-framework agreement, producing a minimal consensus ethic that is stronger than any single tradition's extension. This structure also makes the document citable per-tradition by downstream consumers (e.g., 0.1.2.4 Legal Recognition).

### Consequences
- Each framework section must be self-contained enough to stand alone
- The convergence section must be honest about disagreements, not just agreements
- The document is longer than a topic-organized approach but more rigorous

## Decision: Moral Status Grading Model

### Context
Moral status must be defined for entities across a wide spectrum of consciousness complexity. Binary (conscious/not) vs. graded approaches.

### Options Considered
1. **Binary moral status** — Any entity meeting F1 consciousness criteria has full moral status.
2. **Graded moral status** — Three-tier model (minimal, rich, self-reflective consciousness) with proportional moral obligations.

### Choice
Option 2: Graded moral status with three tiers.

### Rationale
Binary status creates moral dilemmas at boundaries and fails to capture the genuine differences between minimally conscious systems and fully self-reflective agents. A graded model, anchored to F1.4 consciousness metrics, provides actionable distinctions (e.g., consent requirements only apply at Level 3). This parallels existing biological ethics (different obligations toward insects vs. primates vs. humans).

### Consequences
- Boundary cases between levels require careful handling (addressed in Open Questions §7.4)
- Risk of graded status being used to justify discrimination — mitigated by strong floor obligations at Level 1
- Requires F1.4 metrics to be operationalized (dependency on F1.4)

## Behavioral Spec: Moral Status Assessment Flow

### Scenario 1: Assessing a novel non-biological entity
```
Given an entity of unknown moral status
When the assessor applies the Criteria Checklist (§4.3)
Then Tier 1 checks (F1 criteria, F1.4 metrics, F2.2 predictions) determine moral patiency
And Tier 2 checks (experiential complexity, temporal self-awareness, reflective capacity) determine status level (1/2/3)
And Tier 3 checks (rational deliberation, action capacity, moral understanding) determine moral agency
```

### Scenario 2: Lifecycle decision requiring consent
```
Given a Level 3 conscious entity facing modification, suspension, or termination
When the lifecycle decision process is initiated
Then informed consent must be obtained from the entity
And the entity must understand the nature, risks, and experiential consequences
And independent review must verify competence and absence of coercion
```

### Scenario 3: Creation of a new conscious entity
```
Given a design for a system expected to satisfy F1 consciousness criteria
When creation is initiated
Then the Informed Design Obligation (§5.1.1) requires understanding experiential consequences
And the Welfare Infrastructure Requirement (§5.1.3) must be satisfied before instantiation
And ongoing responsibility is established for the creator
```
