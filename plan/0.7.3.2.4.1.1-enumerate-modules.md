# 0.7.3.2.4.1.1 Enumerate Modules [PLAN]

**Description**  
Recursively walk `src/plan-guardian/` and produce a list of every `.js` and `.ts` file, normalizing paths to module identifiers.

**Acceptance Criteria**
1. A script (`scripts/enumerate-modules.js`) walks the directory tree and outputs an array of module paths.  
2. The script excludes test files (`*.spec.js`, `*.test.ts`) and generated artifacts.  
3. The resulting list is written to `tmp/module-list.json`.  
4. The JSON file is syntactically valid and contains no duplicate entries.  
5. The script exits with status 0 and logs a summary count of modules found.
