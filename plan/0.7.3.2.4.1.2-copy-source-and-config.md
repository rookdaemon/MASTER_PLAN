# 0.7.3.2.4.1.2 Copy Source Tree and Configuration [PLAN]

**Description:**  
Copy the full source code tree and configuration files into the staging directory while preserving file permissions.

**Acceptance Criteria**
1. The entire `src/` directory is recursively copied to `build/release-staging/src/` preserving ownership and mode bits.
2. All files under `config/` are copied unchanged to `build/release-staging/config/`.
3. No symbolic links are broken; if links exist they are replicated as links.
4. After copying, a checksum manifest (e.g., SHA‑256) for the copied trees is generated at `build/release-staging/checksums.txt`.
5. The copy step logs its actions to `logs/release-copy.log` and exits with status 0.
