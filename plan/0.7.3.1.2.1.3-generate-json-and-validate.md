# 0.7.3.1.2.1.3 Generate JSON and Validate Schema [PLAN]

Serialize `uniqueExistingPaths` to `inventory-list.json`, ensure UTF‑8 encoding, and validate the file against a JSON schema that requires an array of strings.

**Acceptance Criteria**
1. `inventory-list.json` is written with UTF‑8 encoding and contains the exact ordered array of absolute paths.
2. The JSON file validates against a schema `{ "type": "array", "items": { "type": "string", "format": "uri-reference" } }`; schema failures abort the build with detailed diagnostics.
3. The file size for 10 k entries does not exceed reasonable limits (≈ 2 MB) and is logged.
4. Successful generation logs `inventory-list.json created with N entries` to `logs/parse-inventory.log`.
5. The step completes in ≤ 5 seconds for 10 k entries.
