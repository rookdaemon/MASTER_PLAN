# 0.7.3.2.1.1.1.1 Draft Reviewer Criteria [PLAN]

Write a concise specification that enumerates:
- Required expertise categories (security, compliance, operations).
- Minimum number of reviewers.
- Minimum experience level (e.g., ≥5 years in the relevant domain).

**Acceptance Criteria**
1. The document lists at least one security expert, one compliance officer, and one operations lead.
2. It states a minimum panel size of three reviewers.
3. It includes a required experience threshold (≥5 years) for each reviewer role.
4. The draft is saved to `review-panel/criteria-draft.md`.
5. The draft is ready for stakeholder review within 2 business days of task start.
