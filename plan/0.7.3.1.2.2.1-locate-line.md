# 0.7.3.1.2.2.1 Locate Line [PLAN]
## Description
Search `docs/codebase-map.md` for the line that contains the exact path string `src/plan-guardian/`. Record its line number and current indentation so it can be edited later.

## Acceptance Criteria
1. The line number of the first occurrence of `src/plan-guardian/` is captured.  
2. The full text of the identified line is stored in a temporary variable/file.  
3. If the line is not found, the task fails with a clear error message.  
4. No modifications are made to the file during this step.  
5. The operation is repeatable and idempotent.
