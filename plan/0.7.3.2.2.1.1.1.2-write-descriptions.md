# 0.7.3.2.2.1.1.1.2 Write Field Descriptions [PLAN]
## Description
For each field identified in the previous subtask, compose a concise one‑sentence description explaining its purpose and how it should be populated.

## Acceptance Criteria
1. Every identified field has an accompanying sentence (no missing descriptions).
2. Each description is ≤ 30 words and clearly states the field’s intent.
3. Descriptions are written in present‑tense, active voice.
4. All descriptions are formatted as a Markdown bullet list (field name followed by the sentence).
5. The output passes a spelling/grammar check (simulated by a comment placeholder).
