# 0.7.3.2.1.1.2 Obtain NDAs [PLAN]

Secure signed non‑disclosure agreements from each selected reviewer.

**Acceptance Criteria**
1. A standard NDA template is selected and stored at `legal/nda-template.md`.
2. Each reviewer receives the NDA and returns a signed copy.
3. All signed NDAs are scanned and saved in `review-panel/ndas/` with filenames `<reviewer-name>-nda.pdf`.
4. A checklist confirming receipt of all NDAs is completed and committed.
5. No reviewer is scheduled for the panel until their NDA is on file.
