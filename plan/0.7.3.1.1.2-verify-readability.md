# 0.7.3.1.1.2 Verify Readability [IMPLEMENT]

Validate that each path from `file-list.tmp` points to an existing, readable file.

**Acceptance Criteria**
1. Every path is checked with a filesystem read test.  
2. Any missing or unreadable file is logged to `unreadable.log`.  
3. The process aborts only if more than 5% of files are unreadable; otherwise it proceeds.  
4. A summary report `readability-summary.txt` records total files, successes, and failures.  
5. The step exits with status 0 when the failure threshold is not exceeded.
