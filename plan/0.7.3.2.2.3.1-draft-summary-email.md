# 0.7.3.2.2.3.1 Draft Summary Email [PLAN]

Create the initial draft of the review‑summary email, embedding quantitative results and actionable items.

**Acceptance Criteria**
1. Email draft contains sections: Overview, Count Summary, Action Items Table, Next Steps, Contact Information.  
2. Counts for informational, required‑change, and blocking items are programmatically extracted from `review-results/*.md`.  
3. Action Items Table lists each required change with assigned owner and due date.  
4. Draft is saved as `communications/draft-review-summary.md`.  
5. Draft passes spell‑check and markdown‑format validation (no broken tables or headings).
