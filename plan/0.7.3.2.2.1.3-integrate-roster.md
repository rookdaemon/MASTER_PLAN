# 0.7.3.2.2.1.3 Integrate Criteria into Reviewer Roster [PLAN]

Reference the finalized criteria matrix in the reviewer roster document and ensure the roster uses the scoring rubric for candidate evaluation.

**Acceptance Criteria**
1. The `REVIEWER_ROSTER.md` file includes a “Criteria Reference” section linking to `REVIEWER_CRITERIA.md`.
2. Each listed reviewer entry contains a score column populated using the approved rubric.
3. The roster document is stored in the same `docs/peer-review/` folder.
4. A pull‑request updating the roster is merged with at least one approving review from the governance board.
5. The roster and criteria are both listed in the project’s documentation index (`README.md` or similar).
