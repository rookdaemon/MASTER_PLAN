# 0.7.3.1.1.1.3.1 Write Provenance Build Script [PLAN]

Create the driver script `scripts/build_provenance.py` that merges the plan‑file inventory (`temp/plan_files.json`) and the artifact‑hash data (`temp/artifact_hashes.json`) into a single ordered JSON array and writes it to `artifacts/provenance/provenance.json` with required top‑level metadata.

**Acceptance Criteria**
1. The script reads the two JSON inputs, validates their schema, and merges them into an array sorted alphabetically by `id`.
2. The output JSON includes top‑level keys `generated_at` (ISO‑8601 timestamp) and `generated_by` set to `"plan‑guardian"`.
3. The script writes the result to `artifacts/provenance/provenance.json` and exits with status 0 on success.
4. Running `python scripts/build_provenance.py` creates the file with correct structure and no unhandled exceptions.
5. The script is lint‑ed (PEP 8) and has a module‑level docstring describing its purpose and usage.
