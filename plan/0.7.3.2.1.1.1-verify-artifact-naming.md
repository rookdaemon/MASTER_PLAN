# 0.7.3.2.1.1.1 Verify Artifact Naming [PLAN]

Validate that each required artifact follows the exact naming convention, including version placeholder (`v?`) and correct file extension.

**Acceptance Criteria**
1. The requirements matrix file matches the regex `requirements-matrix_v\d+\.xlsx`.
2. The test‑case mapping file matches the regex `test-case-mapping_v\d+\.xlsx`.
3. The risk‑traceability file matches the regex `risk-traceability_v\d+\.xlsx`.
4. The compliance checklist file matches the regex `compliance-checklist_v\d+\.md`.
5. Any file that does not conform to its expected pattern is reported as a naming violation.
