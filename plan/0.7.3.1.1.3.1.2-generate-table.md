# 0.7.3.1.1.3.1.2 Export markdown – generate markdown table from JSON [PLAN]

Implement the core logic that:

1. Iterates over each deliverable from the loaded JSON.
2. Builds a markdown table row with columns **ID**, **Type**, **Artifact(s)**.
3. Renders each artifact path as a relative hyperlink (e.g., `[artifact1](../artifacts/artifact1.png)`), handling multiple artifacts per deliverable.

**Acceptance Criteria**
- The generated markdown string contains a header row `| ID | Type | Artifact(s) |`.
- Every deliverable from `provenance.json` appears exactly once in the table.
- Artifact links are correctly relativized to the `exports/` directory.
- The function `render_table(data: List[Dict]) -> str` returns the full markdown string.
- Unit tests covering at least: empty list, single deliverable with one artifact, multiple artifacts, and special characters in IDs.
