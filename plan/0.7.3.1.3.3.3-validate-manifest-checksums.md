# 0.7.3.1.3.3.3 Validate Manifest Checksums [PLAN]

Develop automated checks that:
- Parse the manifest’s checksum entries.
- Compare each entry against the SHA‑256 computed from the extracted files.
- Verify that the manifest’s summary section records the archive size accurately.

**Acceptance Criteria**
1. A unit‑test suite (e.g., using pytest) loads a sample manifest and sample extracted files and confirms all checksum matches.  
2. The test fails with a descriptive assertion if any checksum mismatch is detected.  
3. Archive size recorded in the manifest equals the actual file size on disk (± 0 bytes).  
4. The test suite can be invoked with `make test-verify` and exits with code 0 on success.  
5. Test results are shown in CI output, and failures block the pipeline.
