# 0.7.3.2.1.1.1.3 Extract H2 Headings from Map Content [PLAN]
## Description
Parse the content provided by the previous step (the map file) and systematically extract all detected Level 2 headings (H2).
## Scope
- Process the full text content obtained from `docs/codebase-map.md`.
- Identify lines starting with `## ` (two hash symbols followed by a space).
- Extract the text following the `## ` marker.
## Acceptance Criteria
- A clean list of all H2 headings is produced, retaining the verbatim title text.
- The extraction must accurately handle hierarchical markdown structure.
- The output must be structured to allow combination with H1 and H3 lists.
