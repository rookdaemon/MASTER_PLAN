# 0.7.3.1.2.2.2 Apply Edit [PLAN]
## Description
Using the line number and indentation captured in the previous subtask, replace the target line with the same indentation followed by `src/plan-guardian/ → 0.7.3.1`. Perform the write atomically (e.g., write to a temp file then rename).

## Acceptance Criteria
1. The new line exactly matches the original indentation plus the mapping arrow and identifier.  
2. Only the identified line is changed; all other content stays byte‑identical.  
3. The write is performed atomically to prevent partial updates.  
4. A backup of the original file (`docs/codebase-map.md.bak`) is created before the edit.  
5. The edit operation returns a success status only if the file write succeeds without error.
