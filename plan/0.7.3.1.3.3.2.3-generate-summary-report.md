# 0.7.3.1.3.3.2.3 Generate Summary Report [PLAN]

Produce a concise report summarizing acknowledgment completion and share it with the Project Sponsor.

#### Description
- Pull data from the tracking spreadsheet.  
- Calculate overall acknowledgment rate, list outstanding stakeholders, and note any follow‑up actions.  
- Format the report as a PDF and email it to the Project Sponsor and the Documentation Lead.

#### Acceptance Criteria
1. Report includes total number of stakeholders, received acknowledgments, pending acknowledgments, and percentages.  
2. All outstanding stakeholders are listed with responsible owner and next reminder date.  
3. PDF report is generated and attached to an email sent to the Project Sponsor within 7 business days of the original notification email.  
4. Email subject follows the pattern: `Acknowledgment Summary – <Project Name> – <Date>`.  
5. A copy of the report is saved in the project archive under `reports/acknowledgment-summary/`.
