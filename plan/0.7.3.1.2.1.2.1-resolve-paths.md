# 0.7.3.1.2.1.2.1 Resolve Paths [PLAN]

Convert each entry in `rawEntries` to an absolute filesystem path using the known project root.

**Acceptance Criteria**
1. Accepts `rawEntries` (relative or mixed) and produces `absolutePaths` preserving the original order.
2. Uses `path.resolve(projectRoot, entry)` (or equivalent) for conversion.
3. Handles empty or whitespace‑only entries by discarding them and logging a warning.
4. Emits a log entry summarising the number of resolved paths.
5. Completes within 2 seconds for 10 k entries.
