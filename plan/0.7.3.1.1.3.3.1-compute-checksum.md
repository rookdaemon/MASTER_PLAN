# 0.7.3.1.1.3.3.1 Compute SHA‑256 Checksum [PLAN]

Generate a SHA‑256 checksum file for the traceability archive.

### Acceptance Criteria
1. The build script runs `sha256sum` (or equivalent) on the archive located at `archives/traceability/<archive-name>.zip`.  
2. Output is written to `archives/traceability/traceability-checksum.sha256`.  
3. The checksum file contains a single line: `<hash>  <archive-name>.zip`.  
4. The script fails with an error if the archive does not exist.  
5. The checksum file is verified locally (`sha256sum -c`) before proceeding to the next step.
