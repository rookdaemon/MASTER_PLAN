# 0.7.3.1.1.1.1.1 Draft Provenance JSON Schema [PLAN]

Write the initial `schemas/provenance-schema.json` file implementing the required structure.

**Acceptance Criteria**
1. The file exists at `schemas/provenance-schema.json`.
2. The schema defines a top‑level object with a `deliverables` property of type `array`.
3. Items in `deliverables` are objects with required properties:
   - `id` (type `string`)
   - `type` (type `string`)
   - `artifactPaths` (type `array` of `string` items)
4. The schema uses Draft‑07 (or later) `$schema` identifier and includes descriptive `title` and `description` fields.
5. The file passes validation against itself using `ajv validate -s schemas/provenance-schema.json -d test/example-provenance.json` (with a minimal example file).
