# 0.7.3.2.1.1.1.2 Format Extracted Headings into Markdown Outline [PLAN]
## Description
Consume the JSON artifact produced by Subtask 0.7.3.2.1.1.1.1 and render a clean markdown file that mirrors the heading hierarchy using indentations or bullet points to show nesting.
## Acceptance Criteria
1. Generates `artifacts/codebase-map-heading-outline.md` containing only the heading titles, ordered and indented to reflect H1‑H3 levels (e.g., `-` for H1, `  -` for H2, `    -` for H3).
2. The outline omits all body content, images, tables, or code blocks—only headings appear.
3. The file passes a simple validation script that checks:
   - All extracted headings are present.
   - Indentation depth matches heading level (H1 = 0 spaces, H2 = 2 spaces, H3 = 4 spaces).
4. The markdown file is syntactically correct and renders properly in typical markdown viewers.
5. A brief README section in the artifact explains how to regenerate the outline if the source map changes.
