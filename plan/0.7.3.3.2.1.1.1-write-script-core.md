# 0.7.3.3.2.1.1.1 Write Script Core [PLAN]

## Description
Implement the core functionality of `scripts/extract_section_outline.py`:
* Open a markdown file.
* Scan line‑by‑line for top‑level H2 headings (`## `).
* Capture each heading’s exact text and its line number.
* Return a Python list of dictionaries `{ "heading": "## …", "text": "…", "line": N }`.

## Acceptance Criteria
1. A function `extract_h2_headings(md_path: str) -> List[dict]` is defined and returns the correct list structure.
2. Heading detection is case‑preserving and stops at the first two `#` characters followed by a space.
3. The function raises a clear `FileNotFoundError` if the input file does not exist.
4. No external dependencies are required beyond the Python standard library.
5. Unit‑testable (function is pure, no side‑effects).
