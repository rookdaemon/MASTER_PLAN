# 0.7.3.3.2.3.3 Write Justification [PLAN]

## Description
Compose a concise (≤150 words) justification explaining why the selected section and line number constitute the optimal insertion point for the cross‑reference.

## Acceptance Criteria
1. The justification is written in clear, professional language and does not exceed 150 words.
2. It is added to `artifacts/0.7.3.3.2-selected-point.yaml` under key `justification`.
3. The justification references the evaluation criteria from 0.7.3.3.2.2 (e.g., relevance, proximity, readability).
4. The artifact file now contains `section_header`, `line_number`, and `justification`.
5. No modifications to any plan file are performed; only the artifact is updated.
