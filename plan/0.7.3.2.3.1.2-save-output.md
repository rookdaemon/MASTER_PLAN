# 0.7.3.2.3.1.2 Save Output [PLAN]

## Description
Integrate the script from 0.7.3.2.3.1.1 into the CI pipeline so that its output is automatically written to `temp/modules-raw.json`.

## Acceptance Criteria
1. CI step (e.g., in `.github/workflows/ci.yml`) runs the script and redirects output to `temp/modules-raw.json`.
2. The `temp/` directory is created if missing and is added to `.gitignore`.
3. The JSON file is validated (well‑formed) after creation; a failing validation aborts the CI job.
4. The CI job artefact `modules-raw.json` is uploaded for later stages.
5. A fallback log message is emitted if writing the file fails, with clear error details.
