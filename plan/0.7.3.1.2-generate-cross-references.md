# 0.7.3.1.2 Generate Cross‑References [PLAN]

Create bidirectional links between all artifact types and produce a navigable markdown/HTML document.

**Acceptance Criteria**

1. For each requirement, a markdown section contains links to related design decisions, code files, and test results.  
2. For each design decision, backlinks to originating requirements and forward links to implemented code and validated tests are present.  
3. A master `TRACEABILITY.md` in `traceability/plan-guardian/` includes an interactive index (table of contents) with links to every artifact section.  
4. An automated script (`scripts/generate-traceability.py`) can be re‑run to rebuild the cross‑references without manual edits.  
5. The script produces a matching `TRACEABILITY.html` (via pandoc or markdown‑to‑HTML conversion) that preserves all links and is viewable offline.
