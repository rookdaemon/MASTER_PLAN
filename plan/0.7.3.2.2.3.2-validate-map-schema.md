# 0.7.3.2.2.3.2 Validate Codebase Map Against Schema [PLAN]

**Description**  
Run automated validation of `artifacts/codebase-map.yaml` against the organization’s traceability schema and fail the CI pipeline if the map is non‑compliant.

**Acceptance Criteria**
1. A CI job `validate-codebase-map` executes on every PR.  
2. The job uses the official `traceability-schema.json` to validate the YAML.  
3. The job fails with clear error messages when schema violations are present.  
4. On success, a badge `✅ Map schema valid` is posted to the PR.  
5. Validation step is added to the repository’s `.github/workflows/ci.yml`.
