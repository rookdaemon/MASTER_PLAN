# 0.7.3.2.5.2.2 Update Traceability Matrix with Sign‑offs [PLAN]

Edit `docs/traceability-matrix.xlsx` to add a row for each sign‑off, linking it to the relevant feedback items.

**Acceptance Criteria**
1. The matrix file is opened and a new row is added for every sign‑off PDF.  
2. Columns populated: Reviewer, Sign‑off Date (extracted from PDF metadata or reviewer input), Document Link (relative path to `signoffs/0.7.3.2/<reviewer-id>_signoff.pdf`), Feedback IDs Covered (list of IDs).  
3. All existing rows remain unchanged and formatting is preserved.  
4. The updated matrix is saved and committed with a descriptive message (“Update traceability matrix with sign‑offs for 0.7.3.2”).  
5. A checksum (e.g., SHA‑256) of the new matrix version is recorded in the project log.
