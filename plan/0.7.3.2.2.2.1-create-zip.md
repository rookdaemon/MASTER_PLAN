# 0.7.3.2.2.2.1 Create Zip Archive [PLAN]

**Description:**  
Generate a zip archive named `traceability-0.7.3.1-20260409.zip` that contains exactly the files listed in the collection report, preserving their directory hierarchy.

**Acceptance Criteria**
1. The zip command includes all files from the collection report with correct relative paths.
2. No extra files are present in the archive.
3. The archive name matches the pattern `traceability-0.7.3.1-YYYYMMDD.zip` using the current date (`20260409`).
4. The resulting `.zip` file is created in `deliverables/traceability/`.
5. The zip creation command exits with a success status (exit code 0).
