# 0.7.3.2.4.1 Collect Comments [PLAN]

Gather all reviewer comments from the review platform and export them into a machine‑readable format.

**Acceptance Criteria**
1. Exported comment file (e.g., JSON or CSV) contains every comment posted during the review.  
2. Each comment record includes reviewer ID, timestamp, comment text, and reference to the reviewed artifact.  
3. No duplicate or missing comments are detected after a sanity‑check script runs.  
4. The exported file is stored in the designated `feedback/` folder and committed to version control.  
5. A brief checklist confirming successful collection is documented in the task log.
