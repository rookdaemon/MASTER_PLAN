# 0.7.3.1.3.2.2 Copy README [PLAN]

Copy the previously created `README.md` into the staging area for the package archive.

**Acceptance Criteria**
1. The source `README.md` is taken from the repository root (or `docs/README.md` as appropriate).
2. The file is copied to the `build/` staging directory with the exact name `README.md`.
3. File permissions are preserved (readable by all users).
4. If the source README does not exist, the task fails with an informative message.
5. The copy operation is cross‑platform (works on Linux, macOS, Windows).
