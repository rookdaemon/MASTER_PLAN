# 0.7.3.1.1.1.1.1 Implement Collector Script [PLAN]

Create a Python script `scripts/collect_plan_files.py` that:
* Walks the repository tree locating every `plan/*.md` file.
* Parses each file’s YAML front‑matter to obtain `owner` (fallback to `plan-guardian`).
* Uses `git log -1 --format=%cI <path>` to obtain the creation date (`created_at`).
* Emits a JSON array with objects `{id, type:"plan", source, created_at, owner}`.
* Writes the JSON to `temp/plan_files.json`.

**Acceptance Criteria**
1. The script runs from the repository root and exits with status 0.
2. For each discovered plan file the JSON object includes a correct dotted `id` matching the file name.
3. `owner` defaults to `plan-guardian` when the front‑matter field is missing.
4. The output file `temp/plan_files.json` is created and is valid JSON.
5. The script logs a concise summary (total files processed, path to output) to stdout.
