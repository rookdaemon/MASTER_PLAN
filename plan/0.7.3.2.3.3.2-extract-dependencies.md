# 0.7.3.2.3.3.2 Extract Dependencies [PLAN]

## Description
Run a static‑analysis script that parses import statements in each module source file to determine its direct dependencies, then fill the “Direct Dependencies” column of the matrix.

## Acceptance Criteria
1. A script `tools/extract-deps.py` is added (or updated) to read all source files under the codebase and output a JSON map `{module: [dep1, dep2, …]}`.
2. The script is executed and its output is used to update the “Direct Dependencies” column in `docs/codebase-map.md`.
3. All modules have at least one entry (empty list `[]` if none) and the dependency lists are accurate according to the latest code.
4. Changes are committed on the same feature branch `matrix/table-skeleton` with a clear commit message.
5. The script and its version are recorded in the plan file’s artifact section for reproducibility.
