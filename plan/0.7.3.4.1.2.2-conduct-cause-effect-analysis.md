# 0.7.3.4.1.2.2 Conduct Cause‑Effect Analysis [PLAN]

Apply the selected technique to each identified issue, producing detailed cause‑effect artifacts.

**Acceptance Criteria**
1. For every issue identified in the stakeholder interviews, create a cause‑effect diagram using the approved template.
2. Link each diagram to at least two distinct data sources (e.g., interview transcript excerpts, audit logs).
3. Review each diagram with the original interviewee and a subject‑matter expert to confirm accuracy.
4. Record any divergent viewpoints and reconcile them in the diagram notes.
5. Archive all diagrams and supporting evidence in `analysis/results/` with clear naming (e.g., `issue-03-fishbone.pdf`).
