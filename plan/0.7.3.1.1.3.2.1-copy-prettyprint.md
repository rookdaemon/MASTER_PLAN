# 0.7.3.1.1.3.2.1 Copy and pretty‑print provenance JSON [PLAN]

Copy `provenance.json` to `exports/traceability.json` and format it with 2‑space indentation.

**Acceptance Criteria**
1. `exports/traceability.json` is created (or updated) containing the same data as `provenance.json` but formatted with an indentation of 2 spaces.  
2. The file is written with mode `0644`.  
3. If `exports/traceability.json` already exists and its pretty‑printed content matches the newly generated output, the file is not rewritten (idempotent behavior).  
4. The operation logs a clear message indicating whether the file was created, updated, or left unchanged.
