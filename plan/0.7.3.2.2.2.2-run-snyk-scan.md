# 0.7.3.2.2.2.2 Run Snyk Scan [PLAN]

Execute Snyk's vulnerability scanner against the dependency list produced in the previous step.

**Acceptance Criteria**
1. Install Snyk CLI (version ≥ 2023.12) in the CI environment.
2. Feed `artifacts/dependencies.json` to Snyk using its `--file` interface.
3. Capture Snyk output in `artifacts/snyk-results.json`.
4. All findings include CVE ID, severity, and affected version.
5. The CI job fails if any finding has severity **Critical** or **High**.
