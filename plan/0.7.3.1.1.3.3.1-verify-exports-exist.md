# 0.7.3.1.1.3.3.1 Verify export files exist [PLAN]

Ensure that the required traceability export files are present before any git operation.

**Acceptance Criteria**
1. `exports/traceability.md` exists and is a non‑empty file.
2. `exports/traceability.json` exists and is a non‑empty file.
3. Both files are readable by the current user.
4. If either file is missing, the task fails with a clear error message indicating which file(s) are absent.
5. No modifications are made to the files during this verification step.
