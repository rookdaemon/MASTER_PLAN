# 0.7.3.2.2.3.2 Create Signatures [PLAN]

Generate a GPG/PGP detached signature (`*.asc`) for each artifact using the project signing key.

**Acceptance Criteria**
1. A signing script (`scripts/sign-artifacts.sh`) reads the private signing key from a secure location (e.g., `ci/keys/signing.key`).
2. The script creates a detached ASCII‑armored signature (`.asc`) for every file in `artifacts/build/`.
3. All signature files are stored in `artifacts/integrity/`.
4. The script fails if any signature cannot be produced and outputs a clear error message.
5. A log file `artifacts/integrity/signature-generation.log` records the artifact name and the corresponding signature file name.
