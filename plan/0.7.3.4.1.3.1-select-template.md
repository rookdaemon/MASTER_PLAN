# 0.7.3.4.1.3.1 Select and Populate Report Template [PLAN]

Choose an appropriate lessons‑learned report template and fill it with the executive summary, observations, root‑cause analysis, and recommended actions.

**Acceptance Criteria**
1. A vetted template (e.g., organization‑standard lessons‑learned template) is selected and approved by the documentation lead.  
2. All interview findings and root‑cause analysis results are inserted into the appropriate sections.  
3. The report includes at least three concrete improvement actions, each with a clearly assigned owner and target completion date.  
4. The populated draft is saved in the shared draft folder with version `v0.1`.  
5. The draft file path follows the convention `reports/lessons‑learned/0.7.3.4.1.3-draft.md`.
