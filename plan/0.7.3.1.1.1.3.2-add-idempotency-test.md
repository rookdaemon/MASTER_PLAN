# 0.7.3.1.1.1.3.2 Add Idempotency Test [PLAN]

Write a unit test that ensures running the provenance generator twice yields identical output.

**Acceptance Criteria**
1. New test file `tests/provenance.test.js` is added.
2. The test runs `npm run provenance` twice in a temporary directory, captures both `provenance.json` files, and asserts deep equality.
3. The test fails if any differences (including ordering) are detected.
4. The test is included in the CI `npm test` script and causes the CI job to fail on non‑idempotent runs.
5. Test runs complete in under 30 seconds on CI.
