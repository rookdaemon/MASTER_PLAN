# 0.7.3.1.2.2.3 Generate Documents [PLAN]

Using the loaded artifacts and schema, build the traceability markdown document, ensure bidirectional links, write the HTML version, and guarantee idempotent output.

**Acceptance Criteria**
1. The script creates `traceability/plan-guardian/TRACEABILITY.md` with sections for each artifact and cross‑reference links per the schema.  
2. An HTML counterpart `TRACEABILITY.html` is generated (e.g., via `markdown` → `html` conversion).  
3. Running the script twice on the same input yields byte‑identical markdown and HTML files.  
4. A summary log reports: number of sections created, total links generated, and any artifacts missing required links.  
5. The implementation is covered by unit tests verifying link correctness and idempotency.
