# 0.7.3.1.2.2 Compute Hashes [PLAN]

For each file from `inventory-list.json`, calculate SHA‑256, MD5, file size, and creation/modification timestamps.

**Acceptance Criteria**
1. Output a CSV file `file‑metadata.csv` with columns: `path,sha256,md5,size_bytes,created_iso,modified_iso`.
2. All hash values match the file contents when re‑checked with a separate checksum tool.
3. Timestamps are in ISO‑8601 UTC (e.g., `2026-04-09T07:15:00Z`).
4. Files larger than 1 GB are processed using streaming to avoid memory exhaustion.
5. The task logs a summary line: total files processed, total bytes, and elapsed time.
