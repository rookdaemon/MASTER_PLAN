# 0.7.3.2.3.1 Enumerate Modules [PLAN]

## Description
Scan the `src/` directory hierarchy to produce a raw list of every top‑level module directory (i.e., immediate children of `src/`). Capture the module name and absolute repository path.

## Acceptance Criteria
1. A script or CLI command is written that outputs a CSV/JSON list of all top‑level directories under `src/`.
2. The output includes columns: `Module Name` (directory name) and `Path` (relative to repository root).
3. The script runs successfully on the CI environment without manual intervention.
4. The generated raw list is saved to `temp/modules-raw.json` for downstream steps.
5. The output is reviewed by a developer and confirmed to contain *all* top‑level modules (no omissions, no extra entries).
