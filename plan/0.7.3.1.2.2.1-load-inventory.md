# 0.7.3.1.2.2.1 Load Inventory [PLAN]

Read `inventory-list.json` and produce an in‑memory array of absolute file paths to be processed.

**Acceptance Criteria**
1. The JSON file is parsed without errors; malformed JSON causes a clear failure log.
2. The resulting array contains only strings that represent existing files on disk.
3. Files that do not exist are logged as warnings and omitted from the array.
4. The module exports a function `getFileList(): Promise<string[]>` that resolves to the array.
5. Execution time for loading ≤ 5 seconds for inventories up to 100 k entries.
