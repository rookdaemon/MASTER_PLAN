# 0.7.3.2.1.1.1.1 Parse Headings from `docs/codebase-map.md` [PLAN]
## Description
Read the `docs/codebase-map.md` file and programmatically extract every H1, H2, and H3 heading, preserving their order and hierarchical relationship.
## Acceptance Criteria
1. A script or command that reads `docs/codebase-map.md` and outputs a data structure (e.g., JSON or in‑memory list) containing each heading’s text, level (1‑3), and its parent/child relationships.
2. The extraction correctly handles:
   - Multiple consecutive headings of the same level.
   - Skipped levels (e.g., an H3 directly following an H1) by nesting under the nearest higher level.
3. The resulting data structure is stored as an artifact `artifacts/0.7.3.2.1.1.1.1-headings.json`.
4. The script runs without errors on the current repository state.
5. Documentation/comments explain how the hierarchy is derived.
