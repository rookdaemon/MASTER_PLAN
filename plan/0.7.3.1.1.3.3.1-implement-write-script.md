# 0.7.3.1.1.3.3.1 Implement Write‑Index Script [PLAN]

Create the Python script `scripts/write-index.py` that assembles `INDEX.md` according to the specification.

**Acceptance Criteria**
1. The script determines the latest commit hash affecting any file under `traceability/plan-guardian/artifacts/` (e.g., via `git log -n 1 --pretty=format:%H -- artifacts/`).
2. It writes `traceability/plan-guardian/artifacts/INDEX.md` starting with `Latest commit hash: <HASH>`.
3. It appends the exact contents of `traceability/plan-guardian/artifacts/INDEX_TABLE.md` after the header line.
4. It adds a final checklist section containing the line `- [x] No missing references`.
5. The script exits with status 0 on success and prints a concise summary of actions performed.
