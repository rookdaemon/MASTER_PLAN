# 0.7.3.2.2.2.2 Evaluate Candidates Against Rubric [PLAN]

Apply the predefined reviewer scoring rubric to the candidate pool, assign a suitability score to each, and rank them.

**Acceptance Criteria**
1. The scoring rubric is applied consistently to all candidates.
2. Each candidate receives a numeric suitability score (0‑100).
3. Candidates are ranked from highest to lowest score.
4. A short justification (1‑2 sentences) is added for each score.
5. Results are recorded in the same Google Sheet used for gathering candidates.
