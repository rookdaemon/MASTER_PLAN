# 0.7.3.1.2.1.1.1 Read lines from inventory [PLAN]

*Description*  
Open `inventory.txt`, read it line‑by‑line, strip leading/trailing whitespace, and filter out:
  - empty lines
  - lines whose first non‑whitespace character is `#` (comments)

Store the remaining raw strings in an ordered in‑memory array `rawEntries`, preserving original ordering.

*Acceptance Criteria*
1. The implementation uses a streaming read (e.g., `readline` in Node.js, `bufio.Scanner` in Go, etc.) to avoid loading the whole file at once.
2. Whitespace is trimmed from each line before any further processing.
3. Blank lines and comment lines are omitted from `rawEntries`.
4. `rawEntries` length equals the number of non‑blank, non‑comment lines in the source file.
5. The step completes in ≤ 2 seconds for a 10 k‑line file on typical CI hardware.
