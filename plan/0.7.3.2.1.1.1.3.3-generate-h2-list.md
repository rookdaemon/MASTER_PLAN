# 0.7.3.2.1.1.1.3.3 Generate H2 List [PLAN]
## Description
Take the ordered collection of H2 heading texts produced by the previous step and output them as a clean, machine‑readable list suitable for downstream integration.
## Acceptance Criteria
1. The output is a plain‑text list, one heading per line, exactly as extracted (no additional formatting).  
2. The list preserves the original order of headings as they appeared in the source file.  
3. The file `output/h2-headings.txt` (or an equivalent artifact) is created containing the list.  
4. The generation step completes without errors and reports the total count of headings extracted.  
5. The resulting artifact can be consumed by subsequent tasks (e.g., combining with H1/H3 lists).
