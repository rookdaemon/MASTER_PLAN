# 0.7.3.1.1.3.2.1 Create generate‑artifact‑table script [PLAN]

**Description**  
Implement `scripts/generate-artifact-table.py` that reads `artifacts/artifact-list.json` and writes a markdown table to `artifacts/INDEX_TABLE.md` with columns **Category**, **Artifact Name**, **Relative Path**, **Last Commit Hash**.

**Acceptance Criteria**
1. The script exists at `scripts/generate-artifact-table.py` and is executable.
2. It validates that `artifacts/artifact-list.json` is a well‑formed JSON array containing `category`, `name`, and `relative_path` for each entry.
3. It creates `artifacts/INDEX_TABLE.md` and writes a markdown table header row followed by one row per artifact (excluding the commit hash at this stage).
4. On success it prints a concise success message to stdout and exits with status 0.
5. On any error (missing file, malformed JSON, write failure) it prints a clear error to stderr and exits with a non‑zero status.
