# 0.7.3.1.1.3.2 Generate Markdown Table [PLAN]

Consume `artifacts/artifact-list.json` and produce a markdown table with columns: Category, Artifact Name, Relative Path, Last Commit Hash.

**Acceptance Criteria**
1. A script (`scripts/generate-artifact-table.py`) reads `artifact-list.json` and writes `artifacts/INDEX_TABLE.md` containing a markdown table with the required columns.
2. For each artifact, the script obtains the latest git commit hash affecting that file (using `git log -n 1 --pretty=%H -- <file>`).
3. The generated table is sorted alphabetically by Category then Artifact Name.
4. The script exits with status 0 only when the table is successfully written; otherwise it fails with a clear error.
5. CI validates that `INDEX_TABLE.md` exists, is non‑empty, and that every listed Relative Path points to an existing file in the repository.
